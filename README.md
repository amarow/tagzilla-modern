# Scrinia

**Scrinia** is a modern, local-first document management system (DMS) that replaces rigid folder structures with a flexible tagging system. Built with privacy at its core, it features an integrated AI interface, multi-directory scopes, and advanced file previewsâ€”ensuring your sensitive data remains entirely under your control.

## Key Features

*   **Privacy-First:** All data stays local. No cloud dependencies.
*   **Tag-Based Organization:** Organize files with tags instead of folder hierarchies.
*   **Multi-Directory Scopes:** Manage files across multiple source directories seamlessly.
*   **Integrated AI Interface:** Intelligent features to assist with document management.
*   **Advanced Previews:** Preview documents (PDF, Office, Images, etc.) directly within the application.

## Getting Started

To start the application, simply run:

```bash
./start.sh
```

Or, if you prefer using npm:

```bash
npm install
npm start
```
